Meet App

My Meet App project is a PWA made using React and following a test driven development approach that makes use of Google's calendar API. The purpose of the project (completed as part of my course for CareerFoundry) was primarily to introduce concepts like TDD, serverless functions, progressive web apps, and data visualization!

Technology Used: HTML, CSS, JavaScript, React, AWS Lambda, Google API, Jest, Puppeteer, Data Visualization via Recharts.

Check out my Meet App's Github repository!

MyFlix API

This project is the server-side component for the MyFlix client-side (both React & Angular) apps in my portfolio. It provides users with access to information about movies, including their genres & directors. Users are able to sign up, update info, and create/edit a list of favorite movies.

Technology Used: JavaScript, Node.js, Express, MongoDB, Mongoose, Postman.

Check out my MyFlix API's Github repository!

MyFlix Client-Side (React)

This project is the React based client-side component for the MyFlix application based on my existing server-side code (REST API and database). Users can register, login, view movie info (including genres & directors), filter movies, add/remove favorite movies to a user specific favorites list, update user info, as well as delete their accounts entirely.

Technology Used: JavaScript, React, Bootstrap, Redux, Parcel.

Check out my MyFlix React based app's Github repository!

Chat App

This project is a chat app built using React Native. Users can enter a username & select background color prior to entering the chat. In the chat page users can send text messages, choose photos to send from their existing phone library, take new photos (and send), & share their location. The app stores conversations locally and makes use of Firebase Cloud Storage.

Technology Used: JavaScript, React Native, Firebase, Expo, Gifted Chat.

Check out my Chat App's Github repository!
